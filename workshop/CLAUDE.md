# Workshop Module

## Role
HonKit(GitBook fork) 기반 7개 Lab + 3개 Appendix 한국어 워크샵 문서. Physical AI 워크샵의 전체 실습 과정을 안내.

## Key Files
- `SUMMARY.md` -- 목차 (HonKit navigation)
- `book.json` -- HonKit 설정
- `chapters/` -- 마크다운 콘텐츠 (Lab별 챕터)
- `assets/` -- 스크린샷 및 이미지

## Rules
- 모든 문서는 한국어로 작성
- 코드 블록은 복사-붙여넣기 가능하도록 작성
- SUMMARY.md 변경 시 목차 순서 일관성 유지
- 이미지는 assets/에 저장하고 상대 경로로 참조
